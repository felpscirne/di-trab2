# Agount, a Aventura


"Agount, a aventura" é uma pagina web, desenvolvida para a disciplina "Design de Interfaces" no curso "Analise e Desenvolvimento de Sistemas", utilizando recursos de HTML e CSS. Aplicando conceitos de responsividade, para criar uma pagina de boa visualização em diferentes telas. A pagina aplica também conceitos de flexbox e grid conforme vistos em aula.

A pagina web fala consiste em uma wiki com informações divulgativas de uma mesa de RPG jogada por mim e meus amigos, tendo assim um grande afeto pessoal por este projeto, no qual pretendo re-desenvolver utilizando de conceitos mais avançados posteriormente.
